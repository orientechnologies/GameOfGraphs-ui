The download bundle includes:

  • OrientDB 2.2.1-SNAPSHOT, with built-in web server
  • Game of Thrones database
  • The web user interface

To run the application, just do the following:

  • Unzip the file you downloaded
  • Open GameOfGraphs/bin directory
  • Run server.bat (Windows) or server.sh (Linux, OSX)
  • Open Google Chrome (or your favourite web browser) and enter the following URL: http://localhost:2480/GameOfGraphs/index.html
