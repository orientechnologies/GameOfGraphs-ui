Game of Graphs

Game of Graphs is a toy application, developed mostly for fun, to experiment with OrientDB graph database and Vis.js rendering library. In addition, for the front-end I used Angular.js, Bootstrap and FontAwesome

This is Open Source software, released under Apache v2.0 license (see Dataset info). The project has no commercial purpose and is released AS IS, with no warranty. (Please, don't be too critical about the code quality, it started as a toy and I did it in a few hours, literally)

The sample database was created from Game of Thrones wiki (Wikia - just awesome!). The data import required some manual parsing of the wiki pages, so some data are not very accurate.

Thanks to OrientDB LTD for providing the web hosting!
