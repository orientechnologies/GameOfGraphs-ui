Dataset info

The dataset was created parsing http://gameofthrones.wikia.com/ Wiki and loading data into OrientDB v.2.2
You can find the wiki content on this page:

http://gameofthrones.wikia.com/wiki/Special:Statistics (see Database Dumps section)

The source code to import the wiki data into OrientDB as a graph can be found on GitHub
The source code of this website can be found on GitHub as well.

DISCLAIMER: the wiki content was parsed manually and the relationships were loaded based on very basic conventions, so some data can be inaccurate.
